My Books

Posted July 14th 2020

Listed below are pictures and descriptions of some of my favorite books. These books can be both informative and fun to read. if you think you would enjoy reading any of these books and you would like to purchase one, just click on the picture and it will take you to Amazon.com.

The C Programming Language By Dennis M Ritchie and Brian W Kerrighan is an excellent book for both newcomers and expert programmers. The book is packed with many examples, and is yet small and concise. The book is written by the person that created the language, so there is no better source of information. I highly recommend this book to anyone who wishes to become a C programmer.

HTML and CSS is a great book for newcomers to web design. This book is packed with lots of examples, and is easy to read for beginners and challenging enough for experienced programmers and web designers. It is a must read. Click on the image for more details about how you can purchase this book.

Database Systems is my favorite book for learning SQL and the principles of Database design. SQL is the primary language used for talking to a database, and this book explains the language in detail. Whether you are a novice or an advanced SQL developer, you will learn a lot from this book. Click on the image for more details about how you can purchase this book.

Java Programming could not be easier with this book. This was the first book I ever read when it came to learning Java. I can tell you from expereiecne that I have NEVER read a better book that this one when it came to learning Java. Java is a tough language to learn but this book makes it easy. Click on the image for more details about how you can purchase this book.
